# Sorcery City Portal

FiveM RPサーバー「Sorcery City」のポータルサイトです。

## ファイル構成

```
index.html   - メインポータルページ
rules.html   - ルールページ（キーワード収集機能付き）
```

## GitHub Pages での公開方法

1. このリポジトリを GitHub にプッシュ
2. Settings → Pages → Branch: main / root を選択
3. Save → 数分後に公開URL が発行されます

## 注意事項

- 画像はすべてBase64埋め込み済みのため追加ファイル不要
- スタッフ画像（たけのこ・shushu）は未設定のためプレースホルダー表示
- ギャラリー画像（ss1〜ss4）は未設定のためプレースホルダー表示
