Server Rules

このサーバーはルールを緩めに設けています。
ただし、知らなかったでは済まされません。
必ず確認のうえ、街での生活をお楽しみください。

RPから過度に逸脱した行為・他人へのRP強制はBAN対象です

VC必須

サーバーの基本方針

Sorcery City は ロールプレイを楽しむ街です。
ルールは他サーバーより緩めに設定していますが、他プレイヤーのRP体験を著しく損なう行為は厳しく対処します。
不明な点は運営（魔法省）にチケットで問い合わせてください。
をはじめとする運営スタッフが対応します。

  • 他人へのRP強制は禁止。プレイスタイルを押しつけないこと
  • 街全体での会話・行動はRP内で行うこと（OOC発言禁止）
  • キーワード入力後のロール付与は手動のため、お時間をいただく場合があります
  • トラブルが発生した場合は感情的にならず、まずチケットで報告すること
  • ルールは随時更新される場合があります。定期的に確認してください
  • わからないことがあれば や魔法省スタッフへ気軽に相談してください
— ✦ —

六法・ジョブについて

  • 運営に認められている企業の場合は六法での求人が可能
  • 面接は町内で行うこと
  • ジョブの掛け持ちは2つまで可能
  • どちらかのジョブに長期間出勤できない場合、オーナーの意志による解雇が可能
  • PD・EMSなどの公務員職は別途専用ルールに従うこと

🚫 絶対禁止行為（即時BAN）

以下の行為が発覚した場合、即時BAN対応となります。
知らなかったでは済まされません。必ず把握してください。
報告は 宛のチケットか、魔法省チケットで受け付けています。

パワーゲーミング コンバットログ VDM（ビーグルデスマッチ） RDM（ランダムデスマッチ） ゾンビ行為 メタゲーミング グリッチ利用 OOC発言・行動 殺し屋RP セーフゾーン内での犯罪 ルール違反者の肩を持つ行為
— ✦ —

各禁止行為の説明

パワーゲーミング

  • ゲームのシステムや能力を使って、RP的に不自然・不公平な優位を得る行為
  • 例：負傷しているのに全力疾走・車両を盾として異常な使い方をするなど

コンバットログ

  • 戦闘中・シーン中・逮捕直前などにゲームを意図的に落とす行為
  • やむを得ない場合はPD六法に報告し、次回起動時に自首すること
  • シーン中に落ちてしまった場合は など運営にすぐ連絡を

VDM / RDM

  • VDM（ビーグルデスマッチ）：正当なRP理由なく車両で人を轢く行為
  • RDM（ランダムデスマッチ）：正当なRP理由なくランダムに攻撃する行為
  • いずれも即BANとなる最も重いルール違反の一つ

メタゲーミング

  • ゲーム外で得た情報（配信・Discord・SNS等）をRP内に持ち込む行為
  • 例：配信で見た相手の位置情報を利用してゲーム内で行動するなど
  • 配信者への配慮として 🛰マーク の活用を推奨

OOC発言・行動

  • RP中にゲーム外の現実の話題（配信・ラグ・バグ報告など）を会話に持ち込む行為
  • 技術的な問題はチャット機能やチケットで報告すること
  • RP内でのキャラクターとして一貫した行動を心がけること

技術的なルール

ゲームの公平性を保つためのルールです。発覚した場合はBAN対象となります。
不明な場合は魔法省お問い合わせください。

  • ギャング以外の人身ROB禁止

  • PD/EMS車両の譲渡・販売禁止
— ✦ —

犯罪時の服装・乗り物

服装ルール

  • グループで犯罪を行う際は同じ服装で統一すること
  • 上下真っ黒の服装は禁止
  • 指名手配中・シーン中は着替え禁止

乗り物ルール

  • ギャング・ハングレにかかわらず、犯罪時は乗り物の色を統一すること
  • 光沢で色が変わりすぎる乗り物の使用禁止
  • 過度に色味が変わるリバリーの使用禁止
  • メインカラーは全員合わせること

🕵️ 犯罪の基本ルール

犯罪RPは楽しいコンテンツですが、ルールをしっかり守って行ってください。
ルール違反があった場合は 等の運営スタッフが対応します。

基本事項

  • 犯罪時は必ず1つ以上盗むこと（何も取らずに逃走は禁止）
  • レジ金だけ抜いて逃走→発覚した場合強制逮捕・厳重注意の処置
  • 小型・準中型強盗は同ギャング同士でも途中からの介入禁止
  • 警察官に対し銃を構えた・抜いた場合は警察側の反撃が許可される
  • シーン中に使用した車を放置して他の車両に乗り換えることは可能
— ✦ —

シーン・指名手配中のルール

  • 強盗開始から指名手配20分が経過するまでを「シーン」とする
  • 強盗後は30分間指名手配扱い（アナウンスがなくても同様）
  • 指名手配時間が切れても犯人を見失わない限りシーンは続行
  • 準中型強盗以下は指名手配中でも犯罪を起こすことが可能
  • PDが全滅・撤退しても指名手配はかかり続ける

シーン中の禁止事項

  • 鍵が閉められる場所・セーフゾーン・PDが立ち入れない場所への侵入禁止
  • シーン中・指名手配中の着替え禁止
  • シーン中に使用した車両をガレージにしまう行為禁止
  • 指名手配中に自首した場合、刑期と罰金が半減
— ✦ —

歪み（バグ）時の対応

  • 歪み対応はシーンが終了してから行う
  • 犯罪中の歪みで金品が取得できなかった場合は運営から補填あり
  • 補填の条件：主犯の一連のクリップ提出が必須
  • 歪みが起きた際のプレイヤー判断でのやり直しは禁止
  • 必ずチケットで運営に確認を取ること
  • 補填対応は をはじめとする魔法省が確認後に行います
— ✦ —

やむを得ない場合

  • シーン中に夢事情・出勤等でやむを得ず続行不可になった場合でもコンバットログ扱いになる
  • 必ず自首してからログアウトすること
  • 自首も難しい場合はPD六法に報告し、次回起動時に自首の手続きを行うこと
  • 事前に や運営に連絡を入れておくとスムーズです

👤 人質の基本ルール

  • 人質には必ず報酬を支払うこと
  • 同色ギャング・同じ半グレを人質にすることは禁止
  • 出勤中の公務員（EMS・PD）を人質にする行為禁止
  • 人質交渉は誠実に行うこと。過度な要求はPDが拒否できる
  • PDが許容できない条件は引き下がること
— ✦ —

解放条件（2つ選択）

通常人質の解放条件（以下から2つ選択）

車両制限

  • 2分間 車両アタック禁止
  • バイクの禁止

ヘリ制限

  • 1分間ヘリの禁止
  • ※逃走用ヘリを使う場合はヘリ禁止は選べない

交渉系

  • PDが了承できる範囲の解放条件
  • エンタメ要求

注意事項

  • PDが許容できない場合は素直に引き下がること
  • 過度な要求・長引く交渉はNG
— ✦ —

NPC人質の解放条件

  • 1分間の車両アタック禁止
  • 撃ち合い希望
  • PDが了承できる範囲の解放条件
  • エンタメ要求（通常人質と同様）

人質ルールに関する疑問点は または魔法省チケットへ。
シーン中のトラブルは終了後にクリップをつけて報告してください。

🧙 魔法省（運営）へのルール

魔法省はこの街の運営チームです。 をはじめとするスタッフが対応しています。
トラブルや質問は必ずチケットを通して連絡してください。

連絡・報告ルール

  • 報告・通報・質問はすべて【ticket】チャンネルで行うこと
  • ログ・スクリーンショット・クリップがあると対応が早くなります
  • 緊急でない案件は定時瞑想後に対応する場合があります
  • は街全般の管理を担当しています

禁止事項

  • 魔法省職員へのフレンド申請・DM禁止（「サポート以外」と付いた職員を除く）
  • 「サポート以外」と付いている魔法省職員へのメンション・DMは許可
  • 魔法省職員以外が運営・Admin・staffを名乗る行為禁止
  • Admin・運営・staff等の名札を無断で使用することを禁止
  • PD/EMS車両の譲渡・販売・それに関わる行為の禁止
— ✦ —

サポートを受けるには

  • Discordの【ticket】チャンネルからチケットを作成
  • 内容を詳しく記載（状況・日時・関係プレイヤー名など）
  • 可能であればクリップ・スクリーンショットを添付
  • 対応スタッフ（ 等）が確認次第返答します
  • 緊急の場合はチケット作成後、Discordで追記すること

🛰️ 配信について

  • 配信は基本自由
  • 配信中はできるだけ 🛰 マークをニックネームにつけること
  • 衛星（配信）内で起きたトラブルは基本対応しない
  • メタゲーミングの原因になるため、視聴者への過度な情報共有に注意
— ✦ —

🧙 支援・ロール付与について

  • ロールの付与は手動のため時間をいただく場合があります
  • 車・服・家・アイテム等の贈与については詳細が決まり次第アナウンスします
  • 支援に関する質問は または魔法省チケットへ
— ✦ —
